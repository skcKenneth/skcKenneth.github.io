The laboratories connect code to a mathematical question. Each lab should state inputs, units, seed or deterministic settings, expected artifacts, numerical checks, and the command that regenerates its figures.

Suitable technical cases include dengue-grid verification, traffic-wave simulation, interpolation, and stochastic coastal risk.
